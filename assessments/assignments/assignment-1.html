# Assignment 1

For assignment 1 we will build SwiftUI based iOS applications following along with the SwiftUI tutorials. We will do most of the work in class over week 2 and 3.

# Assignment Weight

This assignment is worth 15% of the overall course mark.

# Rubric

  • Complete Building Lists and Navigation

  • Complete Composing Complex Interfaces
  • Complete Working with UI Controls
