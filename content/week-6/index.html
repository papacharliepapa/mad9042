# Challenge: Alternative Mobile Frameworks

Assessment Requirement

This section is a requirement for assignment 3

# Description

Break into teams representing each of the following frameworks and spend 30 - 40 minutes researching your respective framework. Each team will have 5 minutes to speak about the platform to the class. The pitch should include:

  1. A brief description of the framework
  2. Top 3 benefits (pros)
  3. Top 3 dis-benefits (cons)
  4. Ideal scenario to apply the framework
